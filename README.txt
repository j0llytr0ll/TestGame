1. При нажатии на UI кнопку, "Start", на выделенной двухмерной области, создаются случайные координаты на двух осях Х и У. Затем точка от начала начинает перемещаться к этим координатам, с заданной скоростью, и на UI выводится время за которое точка проходит этот путь. При пересечении точкой данных координат, происходит сохранение их в отдельный список и сам список сохраняется в файл формата JSON. При последующем запуске сцены происходит загрузка из файла сохранения и при нажатии на UI кнопку, "Start ", точка сначала перемещается по всем сохраненным координатам и только после этого можно дальше создавать новые координаты и перемещать к ним точку.
2. После того как на двухмерной области были созданы случайные координаты и точка по ним прошла, при наведении курсором и клике на трехмерный объект, находящийся на горизонтальной плоскости, он начинает проходить через все созданные координаты, повторяя путь точки двухмерной области. При первом запуске и клике на сам объект, он так же начнет проходить по всех сохраненным координатам.
3. Так же имеется UI кнопка, "Enemy ", при нажатии на которую, на горизонтальной плоскости создается несколько объектов в случайных координатах, при столкновении с которыми, трехмерный объект останавливается и на UI выводится сообщение с ошибкой. После этого можно навести курсором на этот объект и кликнуть на него, тем самым он удалится и при повторном нажатии на трехмерный объект он продолжит свое движение по всем сохраненным координатам.
